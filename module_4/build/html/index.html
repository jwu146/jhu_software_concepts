class src.order.Order

Represents a pizza order by a customer.

Variables:

  • pizzas (list of Pizza) – List of Pizza objects in the order.

  • cost (float) – Total cost of the order.

  • paid (bool) – Whether the order has been paid.

input_pizza(*ingredients)

Adds a pizza to the order, categorizing ingredients automatically.

Parameters:

ingredients (str) – Ingredients for the Pizza (order doesn’t matter). See Pizza class for ingredient details.

order_paid()

Marks the order as paid.

class src.pizza.Pizza(*ingredients)

Represents a limited-customizable pizza with unordered ingredient types: crust, sauce, cheese, and toppings.

Variables:

  • crust (str) – Select ONE crust type for the pizza (thin, thick, gluten_free).

  • sauce (str) – Select AT LEAST ONE sauce type for the pizza (marinara, pesto, liv_sauce).

  • optional (str cheese,) – Select cheese type for the pizza (defaults to mozzarella).

  • toppings (str) – Select AT LEAST ONE toppings type for the pizza (pineapple, pepperoni, mushrooms).

Raises ValueError if required components are missing.

cost() → int

Calculates the cost of the pizza based on selected options.

Returns:

The total cost of the pizza.
