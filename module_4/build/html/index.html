class pizza.Pizza(*ingredients)

Represents a customizable pizza with crust, sauce, cheese, and toppings.

Variables:

  • CRUST_COST (dict) – Cost associated with each crust type.

  • SAUCE_COST (dict) – Cost associated with each sauce.

  • TOPPING_COST (dict) – Cost associated with each topping.

  • CHEESE (str) – Default cheese type.

  • CHEESE_COST (int) – Cost for cheese (included by default).

  • crust (str) – Selected crust type for the pizza.

  • sauce (list) – List of selected sauces.

  • cheese (str) – Selected cheese (defaults to mozzarella).

  • toppings (list) – List of selected toppings.

cost() → int

Calculates the cost of the pizza based on selected options.

Returns:

The total cost of the pizza.
